This project is develop using Netbean IDE 8.0.2
and using Apache Maven as project management and comprehension tool.

If you want to build and run this project within Netbean build command
please change location of build profile:
    - nbactions-test-offline-modeler-profile.xml
    - nbactions-test-online-modeler-profile.xml
You can also build this project using Maven (recomended)
which will produce 2 jar file:
    - DatacenterVisualization-vCenter-QueryAdapter.jar
    - DatacenterVisualization-Modeler.jar
